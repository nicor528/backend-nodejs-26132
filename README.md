# 🧠 Contenido del curso

Durante el curso se trabajan los siguientes conceptos fundamentales:

### 🌐 Internet y Web
- Arquitectura cliente-servidor
- Protocolo HTTP
- Direcciones IP
- NAT
- Cookies

### ⚙️ Backend con Node.js
- Node.js
- Express.js
- APIs REST
- Rutas
- Middleware
- Manejo de errores

### 🏗 Arquitectura de aplicaciones
- Controllers
- Services
- Models
- Routes
- Separación de responsabilidades

### 🔐 Seguridad
- HTTPS
- SSL / TLS
- Autenticación con JWT
- Access Tokens
- Refresh Tokens

# 🎯 Objetivos del curso

Al finalizar el curso los estudiantes podrán:

- Comprender cómo funciona Internet y HTTP
- Construir APIs REST con Node.js y Express
- Consumir APIs desde un frontend
- Implementar autenticación con JWT
- Comprender conceptos básicos de seguridad web
